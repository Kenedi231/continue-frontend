# Continue App (Front-end)


### First run
Install all dependency
```
flutter pub get
```

### Check flutter
```bash
flutter doctor
```

### Run app with debug mode

```
flutter run --track-widget-creation
```

### Run app
Run emulator or simulator before run app.
```bash
flutter run
```
